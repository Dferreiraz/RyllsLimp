# 🧼 RyllsLimp - Site de Produtos de Limpeza

Este é um projeto de site institucional desenvolvido para uma loja de produtos de limpeza, com foco em apresentar produtos, facilitar o contato com clientes e transmitir confiança.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

---

## 🎯 Objetivo

O objetivo deste projeto é criar uma interface moderna, responsiva e funcional para uma empresa de produtos de limpeza, permitindo:

* Apresentação dos produtos
* Divulgação da empresa
* Contato direto via WhatsApp
* Localização com Google Maps

---

ryllslimp/

---

## ✨ Funcionalidades

* Layout responsivo (desktop, tablet e mobile)
* Navegação com scroll suave
* Botões funcionais (WhatsApp e links externos)
* Integração com Google Maps
* Seções organizadas:

  * Hero
  * Sobre
  * Produtos
  * Entrega
  * Localização
  * Contato
